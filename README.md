## Стек

- **Framework:** Next.js 16
- **Data Fetching:** TanStack Query - Выбрал, потому что здесь уже есть готовая обработка ошибок, кэширование и механизм рефетча. Не пришлось тратить время на написание своих хуков с `useEffect` и `fetch`.
- **Tables:** TanStack Table - Использовал для управления данными. Сортировка, фильтрация и пагинация настраиваются декларативно. Это гораздо надежнее и быстрее, чем писать логику фильтрации массивов вручную.
- **UI Components:** Shadcn UI - Взял за основу, чтобы не верстать каждый инпут с нуля. Это дало аккуратный дизайн и доступность (A11y) сразу.
- **Styling:** Tailwind CSS - Лучший способ быстро накидать стили и при этом не раздувать бандл.
- **Icons:** Lucide React - Потому что это самый легкий и популярный набор иконок для React.

##  Запуск

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/SheeetFace/gaba-test .
   ```

2. **Перейдите в папку проекта:**
   ```bash
   cd <название_папки>
   ```

3. **Установите зависимости:**
   ```bash
   npm install
   ```

4. **Запустите проект:**
   ```bash
   npm run dev
   ```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.
